### 🎯 1. 核心规律（永不过时）
```text

每进制 = 每位 × 进制^位权（从右到左，位权0开始）
二进制：每位 × 2^位权
十进制：每位 × 10^位权
十六进制：每位 × 16^位权（0-9,A=10,B=11,C=12,D=13,E=14,F=15）
```

### ⚡ 3. 转换速成法（3步走天下）
A. 任意进制 → 十进制（乘权求和）
```text

例：0b101101 (二进制) → 十进制
1×2^5 + 0×2^4 + 1×2^3 + 1×2^2 + 0×2^1 + 1×2^0 = 32+0+8+4+0+1 = 45
规律：从右往左，1加2的N次方！
```
```text

例：0x2F9 (十六进制) → 十进制
9×16^0 + F×16^1 + 2×16^2 = 9 + 15×16 + 2×256 = 9+240+512 = 761
规律：F=15，快速×16！
```
B. 十进制 → 二进制（二分法，超快）
```text

例：45 → 二进制
45÷2=22余1, 22÷2=11余0, 11÷2=5余1, 5÷2=2余1
2÷2=1余0, 1÷2=0余1 → 反写：101101
规律：一直除2取余，余数倒序读！
```
C. 十进制 → 十六进制（16分法）
```text

例：761 → 十六进制
761÷16=47余9, 47÷16=2余15(F), 2÷16=0余2 → 反写：2F9
规律：除16取余，余数>9用A-F！
D. 二进制 ↔ 十六进制（4位分组，神技）
```
```text

二进制直接4位一组：
1011 0101 → B5 (查表：1011=B, 0101=5)
反过来：0xB5 → 1011 0101
规律：二进制 nibble(4位)=1位十六进制！
```
